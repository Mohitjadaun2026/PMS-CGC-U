## 🚀 Vision: Empower Every College with a Placement System

Most colleges still manage placements manually or via scattered tools. Our goal is to build a **plug-and-play Placement Management System** that any college can:

- Deploy with ease
- Customize based on their structure
- Use to digitize their **entire placement process**

This is not just a web app — it’s a **framework for institutions** to handle:
- On-campus & off-campus placements
- Real-time student eligibility filtering
- Application tracking per student/company
- Automated communication
- Insightful dashboards for all roles

And it’s **open-source**, so contributors around the world can improve it together.

## ✨ Features

### 👨‍💼 Admin / Placement Cell
- Post jobs (on-campus or off-campus)
- Set eligibility (branch, course, year)
- Auto-filter eligible students
- Track which student applied where
- Upload logos, deadlines, roles, and salary info
- Send automated email notifications
- Access analytics and dashboards

### 👨‍🎓 Students
- Register & manage their academic profiles
- View eligible job listings only
- Apply directly from dashboard
- Track application status
- Get notified via email on relevant postings

### 📬 Companies
- Professional job listing presentation
- Direct application links
- Instant access to filtered student data
