## 🚀 Vision: A Plug-and-Play Placement Portal for Every College

Most colleges still use spreadsheets and manual processes for placements. We're building a system that:

- Can be **easily deployed by any college**
- Handles **on-campus & off-campus placements**
- Allows **admins to post jobs & track applications**
- Lets **students apply, manage, and view status**
- Sends **automated email notifications**
- Offers dashboards for **students & admins**
- Filters eligible students in real-time 🔍

And it’s **100% open-source** so anyone can contribute and scale it further!

## ✨ Key Features

### 👨‍💼 Admin Panel
- Add/edit job postings with detailed info
- Filter students by eligibility (branch/year/course)
- Track which student applied to which job
- Upload company logo, salary, deadlines
- Send bulk emails (Nodemailer integration)
- Dashboard with analytics (upcoming)

### 👨‍🎓 Student Portal
- View jobs based on auto-eligibility
- Apply directly via dashboard
- Track all past & applied jobs
- Receive updates via email

### 📬 Company View
- Professional display of job listing
- Application redirection or resume collection
- Filtered student access (future scope)

## 📦 Installation & Setup

### 🔧 1. Clone the Repository

```bash
git clone https://github.com/Mohitjadaun2026/PMS-CGC-U.git
cd PMS-CGC-U
```

### 📦 2. Setup Backend

```bash
cd backend
npm install
npm start
```

Backend runs at: `http://localhost:5000`

### 💻 3. Setup Frontend

```bash
cd ../frontend
npm install
npm run dev
```

Frontend runs at: `http://localhost:5173`

> 🔁 Make sure both servers are running for full functionality
